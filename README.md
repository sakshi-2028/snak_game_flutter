## 🎮 About

A nostalgic recreation of the classic Snake game — built entirely with Flutter. Eat the food, grow longer, don't hit yourself!

This project is a fun deep-dive into Flutter's rendering, custom layouts, and state management — built with **zero external dependencies**.

---

## ✨ Features

- 🎯 Classic snake mechanics — grow, score, survive
- 📱 **Cross-platform** — Android, iOS, Web, Linux, macOS, Windows
- 🪶 **Zero dependencies** — only the Flutter SDK
- 🎨 Clean Material Design UI
- ⚡ Fast and responsive controls

---

## 🚀 Getting Started

### Prerequisites
- Flutter SDK **3.7+**

### Setup

```bash
# 1. Clone
git clone https://github.com/sakshi-2028/snak_game_flutter.git
cd snak_game_flutter

# 2. Install dependencies
flutter pub get

# 3. Run on any platform
flutter run
```

---

## 📂 Project Structure

```text
lib/
├── main.dart        ← app entry & MaterialApp setup
└── game_page.dart   ← core game logic, board, and controls
```

That's it. Two files. No clutter.

## 🎓 What I Learned Building This

- Flutter's game loop with `Timer.periodic`
- State management for snake position, score, and game-over state
- Handling swipe gestures and keyboard input
- Custom grid-based rendering with widgets
